# Algorithms and Data Structures - Bubble Sort


## Characteristics

- Time complexity:
    - Best: O(n) -> With optimized implementation
    - Average: O(n^2)
    - Worst: O(n^2)
- Space complexity: O(1)
- In-place
- Stable
